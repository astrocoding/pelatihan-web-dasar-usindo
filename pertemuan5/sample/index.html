1. Cara Menjalankan JavaScript

Ada 3 cara untuk menjalankan JavaScript di HTML:

a. Inline JavaScript

JavaScript ditulis langsung di dalam atribut HTML element

<button onclick="alert('Halo!')">Klik Saya</button>

b. Internal JavaScript

JavaScript ditulis di dalam tag <script> di HTML

<script> function internalJSDemo() { alert("Halo dari Internal JS!"); } </script>

c. External JavaScript

JavaScript ditulis di file terpisah (.js) dan di-link ke HTML

<script src="index.js"></script>
ðŸ’¡ Best Practice: Gunakan External JavaScript untuk kode yang kompleks dan dapat digunakan kembali (reusable).

2. Output JavaScript

Buka Developer Console (F12 atau Ctrl+Shift+I) untuk melihat output console!

Cara melihat output:

  • console.log() - Menampilkan di console
  • alert() - Menampilkan popup
  • document.write() - Menulis ke halaman (hati-hati!)
  • innerHTML - Mengubah konten HTML element

3. Lihat File index.js

File index.js berisi contoh lengkap untuk:

  • Variable dan Tipe Data
  • Aturan Penamaan Variable
  • Operator (Aritmatika, Perbandingan, Logika)
  • Control Flow (if-else, switch, for, while, do-while)
  • Fungsi (Function)
  • Array
  • Object
