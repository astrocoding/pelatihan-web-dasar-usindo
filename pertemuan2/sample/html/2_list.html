B. Description List untuk FAQ

Frequently Asked Questions:

Apa itu HTML?
HTML adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web. HTML menggunakan tag-tag untuk menandai berbagai elemen seperti heading, paragraf, link, dan gambar.
Mengapa perlu belajar CSS?
CSS diperlukan untuk mengatur tampilan visual website, seperti warna, font, layout, dan animasi. Tanpa CSS, website akan terlihat polos dan tidak menarik.
Apakah JavaScript sulit dipelajari?
JavaScript memiliki tingkat kesulitan menengah. Untuk pemula, disarankan untuk menguasai HTML dan CSS terlebih dahulu sebelum belajar JavaScript.

Kesimpulan

List dalam HTML sangat berguna untuk mengorganisir informasi. Gunakan:

  • <ul> untuk daftar tidak berurutan (bullet points)
  • <ol> untuk daftar berurutan (numbered list)
  • <dl> untuk daftar definisi (term dan description)

Tip: Anda bisa menggabungkan berbagai jenis list sesuai kebutuhan untuk membuat struktur informasi yang lebih kompleks dan mudah dipahami.
