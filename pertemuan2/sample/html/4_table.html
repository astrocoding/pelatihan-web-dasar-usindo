Kesimpulan

Tag-tag penting untuk tabel:

Tag Fungsi
<table> Wadah utama tabel
<tr> Baris tabel
<th> Header kolom (tebal & tengah)
<td> Sel data biasa

Atribut border:

  • border="0" = Tanpa border
  • border="1" = Border tipis
  • border="2" = Border sedang
  • border="5" = Border tebal
