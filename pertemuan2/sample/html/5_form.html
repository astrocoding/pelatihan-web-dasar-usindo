Catatan Penting

Tag-tag Form:

  • <form> = Wadah formulir
  • <input> = Input data
  • <label> = Label untuk input
  • <textarea> = Text area besar
  • <select> = Dropdown
  • <option> = Pilihan dalam select
  • <button> = Tombol

Jenis Input:

  • text = Text biasa
  • password = Password tersembunyi
  • email = Email
  • number = Angka
  • date = Tanggal
  • checkbox = Kotak centang
  • radio = Pilihan bulat
  • file = Upload file
