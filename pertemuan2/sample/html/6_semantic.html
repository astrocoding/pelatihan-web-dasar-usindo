2. Penjelasan Setiap Tag

A. Header Tag

Contoh Header

Header bisa digunakan untuk bagian atas halaman atau bagian atas section

Biasanya berisi judul, logo, atau informasi pengantar

B. Nav Tag

Navigasi Website:

Tag nav khusus untuk menu navigasi

  • Home
  • About
  • Services
  • Contact

C. Main Tag

Main Content:

Tag main untuk konten utama halaman. Hanya boleh ada 1 main per halaman.

Berisi informasi paling penting dari halaman tersebut.

D. Section Tag

Ini adalah Section

Section digunakan untuk mengelompokkan konten yang berkaitan.

Biasanya section memiliki heading sendiri.

E. Article Tag

Ini adalah Article

Article untuk konten yang berdiri sendiri dan bisa dibaca terpisah.

Contoh: berita, blog post, komentar

F. Aside Tag

Ini adalah Aside

Aside untuk konten sampingan yang mendukung konten utama.

Contoh: sidebar, iklan, link terkait

G. Footer Tag

Ini adalah Footer

Footer untuk bagian bawah halaman atau section.

Biasanya berisi copyright, kontak, atau link tambahan.

Kesimpulan

Keuntungan Semantic HTML:

  • SEO lebih baik - Search engine memahami struktur konten
  • Accessibility - Screen reader bisa navigasi dengan mudah
  • Maintainability - Code lebih mudah dipahami dan dirawat
  • Clean code - Struktur yang jelas dan bermakna

Tag Semantic Penting:

  • <header> - Bagian atas halaman/section
  • <nav> - Menu navigasi
  • <main> - Konten utama (hanya 1 per halaman)
  • <section> - Bagian/grup konten
  • <article> - Konten independen
  • <aside> - Konten sampingan
  • <footer> - Bagian bawah halaman/section
