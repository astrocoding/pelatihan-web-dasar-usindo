Kesimpulan

Tag anchor (<a>) sangat fleksibel dan memiliki banyak atribut:

  • href - URL tujuan
  • target - Cara membuka link
  • title - Tooltip saat hover
  • download - Untuk download file
  • rel - Relasi dengan halaman tujuan
  • hreflang - Bahasa halaman tujuan

Gunakan link dengan bijak untuk membuat navigasi yang user-friendly dan accessible.
