Penjelasan Tag List HTML

Tag Fungsi Kapan Digunakan Contoh
<ul> Unordered List
(List tidak berurutan)		 Daftar tanpa urutan penting <ul><li>Item</li></ul>
<ol> Ordered List
(List berurutan)		 Daftar dengan urutan/langkah <ol><li>Step 1</li></ol>
<li> List Item
(Item dalam list)		 Setiap item dalam UL/OL <li>Isi item</li>
<dl> Description List
(List definisi)		 Pasangan term & definisi <dl><dt>Term</dt><dd>Def</dd></dl>
