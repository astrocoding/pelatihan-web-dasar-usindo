Sampel Button Styling

Contoh styling button dengan berbagai kondisi: Normal, Hover, Focus, dan Active

1. Primary Button

Button dengan warna solid, scale saat hover, dan shadow saat focus

<button class="btn-primary">Primary Button</button>

2. Secondary Button

Button dengan warna berbeda dan efek gerak ke atas saat hover

<button class="btn-secondary">Secondary Button</button>

3. Outline Button

Button dengan border, berubah menjadi solid saat hover

<button class="btn-outline">Outline Button</button>

4. Success Button

Button hijau untuk aksi positif, dengan efek shadow saat hover

<button class="btn-success">Success Button</button>

5. Danger Button

Button merah untuk aksi penting/berbahaya

<button class="btn-danger">Danger Button</button>

6. Rounded Button

Button dengan border-radius penuh (pill shape)

<button class="btn-rounded">Rounded Button</button>

7. Gradient Button

Button dengan background gradient

<button class="btn-gradient">Gradient Button</button>

8. Large Button

Button dengan ukuran lebih besar

<button class="btn-large">Large Button</button>

Catatan Penting

  • Normal: Tampilan default button
  • Hover: Efek saat mouse berada di atas button (gunakan :hover)
  • Focus: Efek saat button mendapat focus dengan keyboard (gunakan :focus)
  • Active: Efek saat button diklik (gunakan :active)

Tips Styling Button

  • Gunakan transition untuk membuat efek smooth
  • Tambahkan cursor: pointer agar jelas bisa diklik
  • Berikan outline atau box-shadow pada :focus untuk accessibility
  • Gunakan transform untuk efek interaktif seperti scale atau translate
  • Pastikan warna text dan background memiliki kontras yang baik
