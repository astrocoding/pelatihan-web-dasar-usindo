1. CSS Variables (Custom Properties)

CSS Variables memungkinkan kita menyimpan nilai yang dapat digunakan kembali di seluruh stylesheet.

Contoh Penggunaan Variables

Primary Color
Secondary Color
Accent Color
:root {
    --primary-color: #3498db;
    --secondary-color: #2ecc71;
    --accent-color: #e74c3c;
    --spacing-unit: 1rem;
    --border-radius: 8px;
}

.box {
    background: var(--primary-color);
    padding: var(--spacing-unit);
    border-radius: var(--border-radius);
}

Linear Gradient

Radial Gradient

background: radial-gradient()

Background Pattern

Multiple backgrounds

Background Properties

/* Solid Background */
.bg-solid {
    background-color: #3498db;
}

/* Linear Gradient */
.bg-gradient {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

/* Radial Gradient */
.bg-gradient-radial {
    background: radial-gradient(circle, #f093fb 0%, #f5576c 100%);
}

/* Background Image */
.bg-image {
    background-image: url('image.jpg');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
}

Display Code Examples

/* Block: Mengambil seluruh lebar */
.display-block {
    display: block;
    width: 100%;
}

/* Inline: Hanya lebar konten */
.display-inline {
    display: inline;
}

/* Inline-Block: Hybrid */
.display-inline-block {
    display: inline-block;
    width: 100px;
    height: 100px;
}

/* Flexbox */
.display-flex {
    display: flex;
    justify-content: space-between;
    align-items: center;
    gap: 10px;
}

/* Grid */
.display-grid {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    gap: 10px;
}

/* Hide Element */
.display-none {
    display: none;
}

Image Styling Code

/* Responsive Image */
.img-responsive {
    max-width: 100%;
    height: auto;
    display: block;
}

/* Rounded Corners */
.img-rounded {
    border-radius: 15px;
}

/* Circle Image */
.img-circle {
    border-radius: 50%;
    width: 150px;
    height: 150px;
    object-fit: cover;
}

/* Image with Shadow */
.img-shadow {
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}

/* Image Filters */
.img-grayscale {
    filter: grayscale(100%);
}

.img-blur {
    filter: blur(5px);
}

.img-brightness {
    filter: brightness(1.2);
}

/* Object Fit */
.img-cover {
    width: 100%;
    height: 300px;
    object-fit: cover;
    object-position: center;
}

Positioning Code

/* Static - Default, mengikuti normal flow */
.pos-static {
    position: static;
}

/* Relative - Relatif terhadap posisi normalnya */
.pos-relative {
    position: relative;
    top: 20px;
    left: 30px;
}

/* Absolute - Relatif terhadap parent positioned */
.pos-absolute {
    position: absolute;
    top: 0;
    right: 0;
}

/* Fixed - Relatif terhadap viewport */
.pos-fixed {
    position: fixed;
    bottom: 20px;
    right: 20px;
}

/* Sticky - Hybrid relative dan fixed */
.pos-sticky {
    position: sticky;
    top: 0;
    background: white;
    z-index: 100;
}

/* Z-index untuk layering */
.layer-front {
    position: relative;
    z-index: 10;
}

Kombinasi Pseudo-Classes

Hover Card

Card ini akan berubah saat di-hover dengan multiple effects

Pseudo-Classes Code

/* Hover - Saat mouse di atas elemen */
.btn-hover:hover {
    background-color: #2980b9;
    transform: scale(1.05);
    box-shadow: 0 5px 15px rgba(0,0,0,0.3);
}

/* Active - Saat elemen diklik */
.btn-active:active {
    transform: scale(0.95);
    box-shadow: inset 0 2px 5px rgba(0,0,0,0.2);
}

/* Focus - Saat elemen mendapat focus */
.input-focus:focus {
    outline: none;
    border-color: #3498db;
    box-shadow: 0 0 0 3px rgba(52,152,219,0.2);
}

/* Link States */
a:link { color: #3498db; }
a:visited { color: #9b59b6; }
a:hover { color: #2980b9; }
a:active { color: #e74c3c; }

/* First-child & Last-child */
li:first-child {
    font-weight: bold;
}

li:last-child {
    border-bottom: none;
}

/* Nth-child */
tr:nth-child(even) {
    background-color: #f5f5f5;
}

/* Not selector */
.item:not(.selected) {
    opacity: 0.5;
}

/* Disabled */
button:disabled {
    opacity: 0.5;
    cursor: not-allowed;
}
