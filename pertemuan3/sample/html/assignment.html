Penugasan CSS Styling

Latihan Praktik untuk Menguasai CSS

Instruksi Umum

  1. Baca setiap tugas dengan teliti
  2. Buat file HTML dan CSS baru untuk setiap tugas
  3. Gunakan CSS Variables untuk nilai yang sering digunakan

Tugas 1

CSS Variables & Color Scheme

Mudah

Objektif:

Membuat sistem color scheme menggunakan CSS Variables

Tugas:

  1. Buat CSS Variables untuk color palette:
    • Primary color (biru)
    • Secondary color (hijau)
    • Accent color (merah/orange)
    • Light, gray, dan dark colors
  2. Buat variables untuk spacing (xs, sm, md, lg, xl)
  3. Buat variables untuk font sizes
  4. Buat 5 button dengan warna berbeda menggunakan variables
  5. Buat 3 card dengan spacing konsisten menggunakan variables

Tips:

  • Gunakan :root untuk mendefinisikan variables
  • Naming convention: --primary-color, --spacing-md
  • Gunakan var(--nama-variable) untuk mengakses nilai

Kriteria Penilaian:

  • Variables didefinisikan dengan benar
  • Naming convention konsisten
  • Warna dan spacing konsisten di seluruh design
  • Mudah untuk mengubah theme dengan edit variables
Tugas 2

Background Styling Showcase

Mudah

Objektif:

Membuat berbagai variasi background styling

Tugas:

  1. Buat 6 section dengan background berbeda:
    • Solid color background
    • Linear gradient (minimal 2 warna)
    • Radial gradient
    • Linear gradient dengan multiple colors (3+ warna)
    • Background pattern menggunakan gradient
    • Background dengan opacity/transparency
  2. Setiap section harus memiliki:
    • Minimum height 200px
    • Teks yang kontras dengan background
    • Border radius 10px
    • Margin antar section

Tips:

  • Linear gradient: linear-gradient(direction, color1, color2)
  • Radial gradient: radial-gradient(shape, color1, color2)
  • Multiple gradients bisa dikombinasikan untuk pattern

Kriteria Penilaian:

  • Semua tipe background berhasil dibuat
  • Warna harmonis dan tidak mengganggu mata
  • Teks tetap mudah dibaca
  • Layout rapi dan responsif
Tugas 3

Typography Portfolio Page

Sedang

Objektif:

Membuat halaman portfolio dengan fokus pada typography

Tugas:

  1. Buat halaman "About Me" dengan elemen:
    • Heading (h1) dengan font khusus dan text-shadow
    • Subheading (h2) dengan letter-spacing
    • Paragraf dengan line-height optimal (1.6-1.8)
    • Quote dengan font-style italic dan border-left
    • List dengan custom styling
  2. Implementasikan:
    • Minimal 2 font family berbeda (heading vs body)
    • Text alignment berbeda untuk tiap section
    • Text transformation (uppercase untuk judul section)
    • Text decoration untuk highlight text

Tips:

  • Gunakan Google Fonts untuk variasi font
  • Line-height penting untuk readability
  • Letter-spacing bagus untuk uppercase text
  • Kontras warna text dengan background minimal 4.5:1

Kriteria Penilaian:

  • Font combination harmonis
  • Hierarchy jelas (heading, subheading, body)
  • Readability excellent
  • Spacing konsisten
Tugas 4

Product Grid Layout

Sedang

Objektif:

Membuat product grid menggunakan Flexbox dan Grid

Tugas:

  1. Buat Header dengan Flexbox:
    • Logo di kiri
    • Navigation menu di kanan
    • Menggunakan display: flex dan justify-content: space-between
  2. Buat Product Grid dengan CSS Grid:
    • Minimal 6 product cards
    • 3 kolom di desktop
    • 2 kolom di tablet
    • 1 kolom di mobile
    • Gap konsisten antar items
  3. Setiap Product Card berisi:
    • Image placeholder (gunakan background color)
    • Product name
    • Price
    • Button "Add to Cart"

Tips:

  • Flexbox: display: flex, justify-content, align-items
  • Grid: display: grid, grid-template-columns: repeat(3, 1fr)
  • Media queries untuk responsive: @media (max-width: 768px)
  • Gap property untuk spacing yang konsisten

Kriteria Penilaian:

  • Header menggunakan Flexbox dengan benar
  • Grid layout responsive di semua screen sizes
  • Spacing konsisten
  • Card design menarik dan konsisten
Tugas 5

Stylish Image Gallery

Sedang

Objektif:

Membuat image gallery dengan berbagai styling effects

Tugas:

  1. Buat gallery dengan 9 image placeholders
  2. Setiap image harus memiliki styling berbeda:
    • 3 images dengan border-radius berbeda (0px, 15px, 50%)
    • 3 images dengan box-shadow berbeda
    • 3 images dengan filter effects (grayscale, blur, brightness)
  3. Implementasikan hover effects:
    • Scale transform
    • Filter berubah (grayscale → color)
    • Shadow menjadi lebih besar
    • Border muncul
  4. Gunakan CSS Grid dengan 3 kolom

Tips:

  • Image placeholder: gunakan div dengan background color gradient
  • Aspect ratio: aspect-ratio: 1/1 atau padding-bottom: 100%
  • Smooth transitions: transition: all 0.3s ease
  • Filter: filter: grayscale(100%)

Kriteria Penilaian:

  • Semua styling effects berhasil diimplementasi
  • Hover effects smooth dan menarik
  • Grid layout rapi
  • Responsive di berbagai screen size
Tugas 6

Positioning Master Challenge

Sulit

Objektif:

Membuat landing page dengan berbagai positioning techniques

Tugas:

  1. Buat Fixed Navigation Bar:
    • Selalu di top saat scroll
    • Memiliki shadow untuk depth
    • Background sedikit transparent
  2. Buat Hero Section dengan:
    • Background image/gradient fullscreen
    • Text centered menggunakan absolute positioning
    • Button positioned di center
  3. Buat Card Section dengan:
    • Badge positioned absolute di corner (NEW, SALE, dll)
    • Icons positioned absolute di berbagai sudut card
  4. Buat Sticky Sidebar:
    • Sticky saat scroll
    • Berisi table of contents atau info
  5. Buat Fixed Footer dengan Social Media Icons

Tips:

  • Fixed nav: position: fixed; top: 0; width: 100%; z-index: 1000;
  • Center absolute: top: 50%; left: 50%; transform: translate(-50%, -50%);
  • Badge: parent position: relative, badge position: absolute
  • Sticky: position: sticky; top: 100px;

Kriteria Penilaian:

  • Semua positioning types digunakan dengan benar
  • Fixed navigation berfungsi sempurna
  • Element positioning akurat
  • Z-index dikelola dengan baik
  • Tidak ada overlapping yang tidak diinginkan
Tugas 7

Interactive UI Components

Sedang

Objektif:

Membuat komponen UI interaktif dengan pseudo-classes

Tugas:

  1. Buat 5 Button Styles dengan hover & active:
    • Primary button (scale on hover)
    • Secondary button (color change)
    • Outline button (fill on hover)
    • Gradient button (shift gradient)
    • Icon button (rotate icon on hover)
  2. Buat Navigation Menu dengan:
    • Link states: normal, hover, active
    • Underline animation on hover
    • Background color on active
  3. Buat Form dengan:
    • Input dengan focus state (border & shadow)
    • Input validation dengan :valid dan :invalid
    • Disabled state untuk button
  4. Buat Card Grid dengan:
    • Hover effects (lift up, shadow)
    • :nth-child untuk alternating colors
    • :first-child dan :last-child styling khusus

Tips:

  • Transitions untuk smooth effects: transition: all 0.3s ease
  • Transform untuk movement: transform: translateY(-5px)
  • Pseudo-class untuk validation: :valid, :invalid
  • Nth-child: :nth-child(odd), :nth-child(even)

Kriteria Penilaian:

  • Semua hover effects smooth dan menarik
  • Active states jelas dan responsive
  • Focus states accessible dan jelas
  • Transitions timing tepat (tidak terlalu cepat/lambat)
  • Visual feedback jelas untuk setiap interaksi
Tugas 8

Final Project: Personal Website

Sulit

Objektif:

Membuat personal website yang menggunakan SEMUA teknik CSS yang telah dipelajari

Tugas:

  1. Setup & Variables:
    • Buat CSS variables lengkap (colors, spacing, typography)
    • Theme variables untuk light/dark mode
  2. Navigation:
    • Fixed/Sticky navigation dengan hover effects
    • Smooth scroll behavior
  3. Hero Section:
    • Gradient atau pattern background
    • Centered text dengan typography styling
    • CTA buttons dengan hover animations
  4. About Section:
    • Flexbox layout untuk image & text
    • Typography hierarchy jelas
    • Profile image dengan styling (border-radius, shadow)
  5. Skills/Services Section:
    • CSS Grid untuk cards
    • Hover effects pada cards
    • Icons dengan positioning
  6. Portfolio/Projects Section:
    • Image gallery dengan filters
    • Hover overlay effects
    • Badge positioning (featured, new, dll)
  7. Contact Section:
    • Form dengan focus & validation states
    • Background styling menarik
    • Social media links dengan hover effects
  8. Footer:
    • Background gradient atau pattern
    • Links dengan pseudo-classes

Requirements:

  • ✅ Menggunakan CSS Variables
  • ✅ Minimal 3 tipe background styling berbeda
  • ✅ Typography hierarchy jelas
  • ✅ Flexbox DAN Grid digunakan
  • ✅ Image styling dengan effects
  • ✅ Minimal 2 tipe positioning (fixed, absolute, atau sticky)
  • ✅ Hover effects di minimal 5 elements
  • ✅ Focus states untuk form elements
  • ✅ Responsive design (desktop, tablet, mobile)
  • ✅ Smooth transitions & animations

Kriteria Penilaian:

  • Functionality (30%): Semua fitur berfungsi
  • Design (30%): Visual menarik dan profesional
  • Code Quality (20%): Kode rapi, terorganisir, dengan comments
  • Responsiveness (10%): Bekerja di semua device
  • Creativity (10%): Ide original dan implementasi unik

Bonus Points:

  • Implementasi dark mode toggle (CSS variables)
  • Advanced animations menggunakan @keyframes
  • Custom scrollbar styling
  • Accessibility features (focus visible, semantic HTML)
  • Performance optimization (minimal CSS)
